Raincoat CSS - base styles

Often, we reach out for a framework to do a simple task. Bootstrap, Bulma, Tailwind - they all are great in their way, but still too much for some things. For the basic page, I have created Raincoat. It is a small, opinionated set of styles, built to get you a swift start where you need it.

Life is really simple, but we insist on making it complicated.
~ Confucius

This quote expresses a lot of my own thought process. It is essentially the same reason why I enjoy working with Svelte or Vue a lot more than with React. No hate there, React is cool in it's own pure JS way. But is this really necessary? Why not using sensible abstractions to remove the clunky parts, and make your own code more slick?

The Raincoat CSS framework should be easy to use. Edit the _config.scss, and the fun begins.
